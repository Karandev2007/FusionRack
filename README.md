<h3 align="center">Zentra / FusionRack</h3>

This is a modern design 2-piece server tower that features total of 4 in-built cooling fans with speed and direction control for more customization, and removable magnetic lid covers for front side.

## Features:
- In-built Cooling System with speed and direction control.
- Removable magnetic lids for easy hardware access.
- Designed for modern era
- Easily assemble or disassemble.
- 2 USFF or TFF mini pc space & a extra plate to attach as you wish for other broads.
